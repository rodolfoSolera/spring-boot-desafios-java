# Spring Boot Desafios Java

Repositório criado para armazenar desafios, testes técnicos e projetos práticos utilizando Java e Spring Boot.

## Objetivo

Praticar desenvolvimento backend com foco em:

* Java
* Spring Boot
* APIs REST
* Boas práticas
* Arquitetura backend
* Testes
* Docker
* Banco de dados

---

## Desafios

### 1. Desafio Itaú

Implementação de uma API backend baseada em um desafio técnico do Itaú.

#### Tecnologias utilizadas

* Java
* Spring Boot
* Maven

spring-boot-desafios-java/

## Como executar um projeto

```bash
# clonar repositório
git clone https://github.com/rodolfoSolera/spring-boot-desafios-java.git

# entrar na pasta do desafio
ex: spring.boot.desafio.itau

# executar projeto
./mvnw spring-boot:run
```
